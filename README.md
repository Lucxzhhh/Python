# 📘 Sobre este repositório

Este repositório foi criado com o objetivo de documentar minha evolução em Python, refazendo um curso que já estudei anteriormente, porém agora com uma abordagem mais avançada e consciente.

Dessa vez, o foco não é apenas concluir as aulas, mas entender profundamente os conceitos, praticar com mais autonomia e registrar aprendizados importantes ao longo do caminho.

Além dos códigos, este repositório também serve como um espaço de anotações, onde organizo ideias, explicações e pontos que considero relevantes para fixação do conteúdo.

# 🚀 Objetivo
Reforçar a base em Python
Aprofundar conceitos já estudados
Melhorar a organização de código e boas práticas
Criar um histórico real de evolução
Desenvolver consistência nos estudos
## 🧠 Observação

Este repositório não representa apenas resultados finais, mas também o processo de aprendizado. Por isso, podem existir códigos simples, testes e experimentos — todos fazem parte da construção do conhecimento.
